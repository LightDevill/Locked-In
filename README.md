# Us App (Android-first React Native with Expo SDK 48)

Production-oriented starter for **Us**, a private couple companion app for exactly two users.

## Setup
1. `cd us-app`
2. `cp .env.example .env`
3. Add Supabase keys
4. `npm install`
5. `npx expo start`

## Android APK build
- `eas build --platform android --profile preview`
